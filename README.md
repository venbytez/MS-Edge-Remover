# MS Edge Remover

Batch script for Windows 10/11 to remove Microsoft Edge, clean leftovers, apply install/update blocks, and optionally enable a startup guard.

## Files

- `EdgeRemover.bat` - Main tool (remove, block, guard, quick verify)
- `Verify.bat` - Optional standalone health check

## Quick Start

1. Run `EdgeRemover.bat` as Administrator.
2. Choose option `1` for full cleanup and protection.
3. Reboot.
4. Run quick verify from menu (or `Verify.bat`).
